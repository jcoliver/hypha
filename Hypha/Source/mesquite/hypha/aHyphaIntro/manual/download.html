You can download and install Hypha in one of two ways:

  1. Start Mesquite 2.75 or later. Choose File>Check Now for Notices/Installs. In the dialog box that is presented, enter http://mesquiteproject.org/packages/hypha/install.xml, and press OK.
  2. Download the package from this page by clicking on the link below, unzip the file downloaded, and move the hypha directory into your Mesquite_Folder/mesquite/ directory (see image below).

  • Hypha Instructions (pdf)

If you previously downloaded either the base system of Mesquite or Hypha, and intend to download a new version of Hypha, you should download new versions of all packages to ensure compatibility. Hypha requires Mesquite 2.75 or later.
